Escrevendo um parágrafo em HTML

Esta Tag define um parágrafo para um texto escrito em uma página
 html podendo este conter apenas uma ou mais linhas. Perceba também
que utilizamos a tag "br" antes do paragráfo, para pular linha do trecho
anterior para este e também na produção do texto para pular as linhas.
Há, um outro detelhe, veja que utilizamos a tag "strong" para grifar e, negrito.
