Switch Case

O switch case é uma estrutura condicional que serve para testar uma variável ou expressão contra vários valores possíveis. É como se fosse um "menu de opções": você compara um valor e executa um bloco de código específico dependendo do caso que combinar. Ele funciona assim: O valor passado no switch é comparado, um a um, com cada case. Quando um case corresponde ao valor, o código dentro dele é executado. Normalmente usamos break; no final de cada case para parar a execução (senão ele continua para o próximo caso). O default é opcional e serve como um "plano B" (executado se nenhum case for verdadeiro).
