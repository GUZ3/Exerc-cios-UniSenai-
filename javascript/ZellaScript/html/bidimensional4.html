VETOR

Bidimensional

Um vetor bidimensional em JavaScript é, na prática, um array de arrays. Enquanto um vetor unidimensional armazena valores em linha (um após o outro), um bidimensional organiza dados em linhas e colunas, como se fosse uma tabela ou uma matriz.
