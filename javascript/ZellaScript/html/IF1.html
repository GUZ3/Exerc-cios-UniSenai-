ESTRUTURAS CONDICIONAIS

IF/ELSE IF/ELSE

A estrutura condicional SE é um recurso utilizado na programação para permitir que o programa tome decisões. Ela avalia uma condição lógica, que pode ser verdadeira ou falsa. Se a condição for verdadeira, o programa executa um conjunto de comandos específicos. Caso a condição seja falsa, o programa pode simplesmente não executar nada ou seguir um caminho alternativo, utilizando o SENÃO. Esse tipo de estrutura é fundamental porque permite que o programa siga diferentes fluxos de execução de acordo com as situações, tornando-o mais dinâmico e inteligente.
