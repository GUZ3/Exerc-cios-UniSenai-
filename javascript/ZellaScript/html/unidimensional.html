VETOR

Unidimensional

Um vetor unidimensional em programação é uma estrutura de dados usada para armazenar uma sequência de elementos do mesmo tipo (números, caracteres, objetos, etc.) em uma única linha contínua de memória.

Ele é chamado de unidimensional porque possui apenas uma dimensão, ou seja, uma única forma de organização: cada elemento é acessado por meio de um índice (posição dentro do vetor).

Um vetor de 5 números inteiros poderia armazenar [10, 20, 30, 40, 50].

O primeiro elemento estaria na posição 0 (ou 1, dependendo da linguagem), o segundo na posição 1, e assim por diante.
